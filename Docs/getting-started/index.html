Introducing Simple.css

Welcome to Simple.css. A framework built to be simple! This project was made so that people who don't like linking tens to thousands of CSS frameworks to only link one lightweight but powerful framework.


A little FAQ


What is CSS and what is the problem with it?

Cascading Style Sheets are style documents for Websites. Usually websites have CSS Code written to them to make them look unique or have more funtionality, also some websites use a lot of CSS files. These CSS files can make slow or even unresponsive websites. Simple changes this by making one light stylesheet with incredable performance.


How is Simple.css different from frameworks like Bootstrap or W3?

Simple is based on Bootstrap, W3, and Font Awesome. Meaning that you don't need to write down a ton of code, because the code you already know just works, it's that simple.


What is AT.css?

AT.css is a stylesheet that brings themes and effects like shading. This CSS also brings vibrant colors like Ex Green Lite Red and Areo Blue, while keeping more dark and foucused colors like Metro Red Metro Green and Dark Yellow. Increasing transparency while making things stand out.
