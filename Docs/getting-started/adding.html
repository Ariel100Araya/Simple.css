Adding Simple.css to your website

Ariel Araya | Developer

Lets begin


To begin adding Simple.css to your website project, you first need to find the folder and copy the installed version of Simple.css (If you need to download it, click here) to your website folder and then add the link tag, inside the link tag you have to add rel="stylesheet" href="Simple.css if it is in a subfolder, make sure to add that. Once you save, you are done! Simple.css has been added to your website!


Note: you can also add a link tag to our website hosting Simple.css and can get the same result. For more info about this, click here
