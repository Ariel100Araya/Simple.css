Adding Simple.css to your website using URL

Ariel Araya | Developer

Notice! This is a subarticle. Context can be understood better if you read this article first.


Although it may load faster if the website is stored locally, updates will require you to replace Simple.css. To help with this issue, we are hosting Simple.css on this website. All that you have to do is follow the instructions on the head article and instead of adding src="Simple.css", you will add src="https://www.ariel100araya.github.io/Simple.css/Simple.css" (Note: The link is case sensitive)
