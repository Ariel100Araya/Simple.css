Base

One of the new features in Simple.css is themes!


The transparency tag

<!DOCTYPE html>
<html>
<head>

</head>
<body>

<h1 class="OP[enter level here]">This is a Heading</h1>
<p class="OP[Enter level here]">This is a paragraph.</p>

</body>
</html>

Transparency works on images too! You can add this tag and it will make your image transparent.

<img class="OP[enter level here]" src="exampleimage.png"></img>

Examples

Here is an example with text. The heading will be more transparent because it has the 0.5 tag

<!DOCTYPE html>
<html>
<head>

</head>
<body>

<h1 class="OP05">This is a Heading</h1>
<p class="OP1">This is a paragraph.</p>

</body>
</html>
